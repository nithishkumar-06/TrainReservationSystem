Train Reservation System

A basic level Console based Train Reservation system with Python, with database connection to store locally with MySql.

Functions:
1) Login/Register with User Validation 
2) Book tickets with desired Berth.
3) Cancel Tickets 
4) RAC and Waiting tickets were moved to cancel tickets.
5) View booked tickets as per the user.
6) View Available Tickets.

Future Works and Add-ons:

1) Implementation of UI with Django Framework.
2) Try to host as container in any cloud platform.
